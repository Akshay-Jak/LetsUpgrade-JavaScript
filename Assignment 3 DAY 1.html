DATA TYPES IN JAVA SCRIPT

Like anyother programming language Java Script also has a set of data types. There are eight basic data types Java Script is called dynamically typed i.e. data types exist but variables are not bound to them.

1.NUMBER

The number represents both integer and the floating point numbers . There are many operations in this data type.

2.BIGINT

This data type is for representing integers of arbitary length.This type is used when we need big numbers i.e. for cryptography or microsecond precision timestamps.

3.STRING

Like anyother programming language a string Java script is the same. A string must be denoted between a single or a double quote

4.BOOLEAN

The boolean type has only two values True and False.

5.NULL

In Java script null is not a 'null pointer' or a 'reference to a non-existing object',but is a special value which represents 'nothing',its just an empty value.

6.OBJECTS AND SYMBOLS

Object is a special type that are used to store colllection of data and more complex entities. Symbols are used to create a unique identifiers for objects.We have to mention it here for the sake of completeness, but can also postpone the details till we know obejcts.

7.TYPEOF OPERATOR

The typeof operator returns the type of the argument.It is useful when we want to process values of different types differently or just want to do a quick check. It can be written as typeof x-operator (or) typeof(x)-function.

8.UNDEFINED VALUE

Its a special value like null.It makes its own type.It means 'the value is not assigned'
